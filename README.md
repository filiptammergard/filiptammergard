## Open Source

I'm passionate about open source. Here's a list of some of the open source projects I'm involved in:

- [React-TypeScript Cheatsheet](https://github.com/typescript-cheatsheets/react): A cheatsheet that helped me a lot when I started working with React together with TypeScript. I'm helping maintain and improve it.
- [namedays](https://github.com/filiptammergard/namedays): A minimal API for namedays I created. Look them up by name, date, country, or ID. Open for anyone to use, give feedback on and contribute to!
- [text-analyzer](https://github.com/filiptammergard/text-analyzer): A small library to analyze text—count characters, words, sentences, paragraphs, and reading time.
- [robber](https://github.com/filiptammergard/robber): Convert to and from rövarspråket, a Swedish language game.
- [roman](https://github.com/filiptammergard/roman): Convert to and from Roman numerals.
